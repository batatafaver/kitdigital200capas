# **App Name**: Notebook Covers

## Core Features:

- Cover Display: Showcase a diverse array of notebook cover designs, easily browsable by the user.
- Template Customization: Provide access to customizable templates on Canva with editable text, image uploads, and repositioning of graphic elements.
- Preview Generation: Generate a real-time preview to help users to see their final customized cover.
- Instant Download: Enable immediate downloads of purchased customized covers.

## Style Guidelines:

- Primary color: Soft Sky Blue (#A7D9ED) for a bright and friendly feel.
- Background color: Light Lavender (#F9F3FF) - desaturated, bright tint for a calming background.
- Accent color: Vivid Pink (#E91E63) for high contrast on key action elements and text.
- Body and headline font: 'Poppins', a geometric sans-serif.
- Use clean, vector-based icons. These need to be minimalistic to keep a modern look.
- Maintain a clear, simple grid layout with generous spacing for easy navigation.
- Incorporate subtle transitions, like fade-ins or smooth scrolling.
